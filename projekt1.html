Googles Teachable Machine

Mit Googles *Teachable Machine* kann man ohne Programmierkenntnisse eigene maschinelle Lernmodelle erstellen. Es bietet die Möglichkeit, Modelle für die Erkennung von Bildern, Geräuschen oder Körperhaltungen zu trainieren, indem man eigene Daten hochlädt oder direkt über die Webcam oder das Mikrofon aufnimmt. Das Tool eignet sich ideal, um schnell neue Ideen zu testen, kreative Projekte zu verwirklichen oder den Umgang mit maschinellem Lernen zu erlernen.

Ich habe die Bilderkennungsfunktion für mein Projekt DrinkalotAI verwendet. Meine Idee war es einen Arbeitsassistenten zu bauen, der den Benutzer daran erinnert wärend seiner Zeit am Compter genug zu trinken.
Das ganze sollte funktionieren, indem eine Trinkflasche von einer externen Webcam erfasst wird und dann mit der Hilfe der Teachable Machine der Füllstand kontrolliert wird. Der Benutzer soll dann daran erinnert werden regelmäßig zu trinken.

Damit dies funktioniert muss zunächst die Teachable Machine von Google trainiert werden. Es handelt sich dabei um einen Prozess, bei dem Verschiedene Klassen mit Beispielinformationen bestückt werden. Im Zuge dieses Projektes wurde die Machine mit Bildinformationen gefüttert. Die verschiedenen Klassen sind benannt von 0% bis zu 100%. Die eingespeisten Bilder zeigen die Flasche mit dem jeweiligen Füllstand an. Jede Klasse wird mit bis zu 200 Bildern bestückt, welche sich durch verschiedene Lichteinfallswinkel unterscheiden, damit es nicht zu falschen Ergebnissen komt.
Um weitere Messfehler zu verhindern, befindet sich die Messflasche in einem festen Setting mit einem fixen Abstand zur Webcam. Der Benutzer muss nun nur noch die Flasche auf den dafür vorgesehenen Platz stellen.

Damit ist die Hardware und das Teaching erklärt, aber anwendbar ist es so noch nicht.
Googles Teachable Machine hat die Funktion, trainierte Modelle zu exportieren und in einen Code mit einzubauen.
Somit kann man recht simpel aus dem Modell eine Website bauen. Die Website für dieses Projekt besteht aus einer Landingpage mit einer kurzen Einführung. Von dieser Seite aus kommt man dann per Startknopf auf die Seite, in der die Teachable Machine eingebunden ist. Auf dieser Seite wird dann Der Füllstand als Prozentwert angezeigt.
Sobald der Benutzer fertig ist, drückt er auf den End Button wodurch die dritte Seite erscheint, auf dem gratuliert wird. Von Hieraus kann DrinkalotAI auch neu gestartet werden.

Ein Problem was bei diesem Projekt auftritt ist allerdings, dass es nur in dem festen, trainierten Setting funktioniert, und somit trotz einer, von allen erreichbaren Website, nicht von jedem anwendbar ist.
Zudem ist der Sinn fragwürdig, da man sich eben so gut die Flasche anschauen kann, wie DrinkalotAI.

Ein Punkt an dem man auf jeden Fall noch weiter machen kann, wäre eine Timerfunktion, wodurch man auch wirklich regelmäßig daran erinnert wird etwas zu trinken, sowie eine Art Endstatistik, aus welcher man entnehmen kann, wie viel tatsächlich getrunken wurde.
Bei dem Versuch dies umzusetzen bin ich selber an meine Grenzen gestoßen, da ich nicht herausgefunden habe, wie ich die von der Teachable Machine erkannten Werte/Klassen weiter verwenden kann. Dies wäre notwendig gewesen, um einen Counter oder andere Klassenabhängige Funktionen hinzuzufügen.

Das Projekt ist in Form einer Website unter https://drinkalotai.netlify.app/ zu erreichen.
